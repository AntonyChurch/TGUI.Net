TGUI.Net is an easy to use cross-platform GUI for SFML.Net.
This project is a port of the C++ TGUI library.

There are no tutorials yet, but you should take a look at the example code.
You can also find instruction on how to setup TGUI.Net for windows and linux. It hasn’t been tested on Mac yet, but it should by similar to using it on linux.

Click here to download TGUI.Net (latest development snapshot).
But please keep in mind that this is a very new library so there might still be minor issues.
