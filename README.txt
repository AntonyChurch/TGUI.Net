TGUI - Texus's Graphical User Interface
=======================================

TGUI.Net is an easy to use GUI for SFML.Net.
For more information, take a look at the website (http://tgui.net).



Download
--------

You can download the latest release on Github (https://github.com/texus/TGUI.Net).



Tutorials
---------

Linux:   http://tgui.net/linux/
Windows: http://tgui.net/visual-studio/
